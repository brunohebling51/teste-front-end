Teste Front-end - Econverse

Desenvolvido por Bruno Hebling

- Testes

Este projeto não possui testes automatizados.

- Estilização

O projeto utiliza SCSS (Sass) para organização dos estilos, seguindo uma estrutura modular por componentes.

- Funcionalidades
Listagem de produtos via API
Modal de produto
Carrossel de produtos
Navegação por categorias

Projeto desenvolvido como parte de um teste técnico para vaga de Front-end.

---

Tecnologias utilizadas

- React
- TypeScript
- Vite
- SCSS

---

Instalação

Clone o repositório:

acesse o terminal ou git bash e execute para clonar o repositório.

git clone https://github.com/brunohebling51/teste-front-end.git

Acesse a pasta do projeto:

cd teste-front-end

Instale as dependências:

npm install

Rode o projeto:

npm run dev

em seguida acesse http://localhost:5173 ( irá aparecer no próprio terminal o local caso de erro )
